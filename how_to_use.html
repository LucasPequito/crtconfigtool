Resolution Refresh Tool

To calculate the desired resolution (Only vertical pixel count is needed) or refresh rate, First you need to fill the mandatory Horizontal scanning frequency field (remeber the KHz to Hz convertion achieved by multiplying x1000). After that, fill the remaining fields leaving the inquired field empty (if you need to know the refresh rate, fill vertical pixel count, if you need to know the vertical pixel count, fill refresh rate).

After you've finished filling the necessary information, all you gotta do is press on the >RUN button, and the empty field will be completed.

Aspect Ratio Tool

To calculate the aspect ratio of a given resolution, or the pixel count of a certain aspect ratio, fill all spaces leaving one empty for the incquired field. Keep in mind that both aspect ratio fields count as 1 field, meaning they both have to be filled for it to count as "filled".

This tool is able to operate on a mixed mode, in which you can insert one of the resolution dimensions (px) and the desired aspect ratio. By pressing run the algorithm will generate the other resolution dimension.
